# Labo 1 — Liste préliminaire de pièces électroniques

## Matériel pour l'installation initiale du Raspberry Pi
- **Raspberry Pi 5** comme station de programmation distante.
- **Clavier** et **écran tactile** pour la configuration locale.
- **Alimentation USB-C** officielle/équivalente pour le Pi 5.
- **Carte microSD 64 GB** préchargée avec Ubuntu Server.
- **Câble micro-USB** pour le clavier (si requis).

## Connectivité et accès distant
- **Connexion Ethernet** pour la configuration réseau de base.
- **Cloudflare Tunnel (cloudflared)** installé sur le Pi pour l'accès SSH sécurisé.
- **Réseau WiFi** prêt pour un deuxième profil de connectivité au Cégep.

## Programmation et périphériques IoT
- **LilyGO A7670G** (ESP32 + modem LTE + GPS) avec **antennes LTE et GPS**.
- **Carte SIM activée** avec code PIN désactivé ou connu.
- **Câble USB-A vers USB-C** pour flasher et alimenter le LilyGO depuis le Pi.

## Prototype breadboard minimal
- **Plaquette de prototypage** (breadboard).
- **2 LEDs 5 mm** (rouge et verte) avec **2 résistances 220 Ω ou 330 Ω**.
- **Fils jumper** pour les connexions GPIO.

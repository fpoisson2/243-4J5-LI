# Plan de séquence - Semaine 07

## Titre
Remise projet de mi-session + Introduction LoRa/Meshtastic

## Objectifs de la semaine
- Remettre le projet de mi-session (schéma, PCB, documentation)
- Découvrir la technologie LoRa et le protocole Meshtastic
- Prendre en main la carte LilyGO T-Beam SUPREME
- Configurer les premiers paramètres du réseau mesh

## Contenu théorique

### Introduction à LoRa
- Qu'est-ce que LoRa (Long Range)?
- Modulation CSS (Chirp Spread Spectrum)
- Caractéristiques : portée, débit, consommation
- Bandes de fréquences (915 MHz en Amérique du Nord)

### Meshtastic
- Protocole de réseau mesh décentralisé
- Architecture peer-to-peer
- Cas d'utilisation : communication hors réseau
- Comparaison avec d'autres technologies (WiFi, LTE)

### T-Beam SUPREME
- Caractéristiques de la carte
- Module LoRa SX1262
- GPS intégré
- Interface utilisateur et configuration

## Activités

### Théorie (2h)
- Présentation de LoRa et Meshtastic
- Démonstration d'un réseau mesh simple
- Architecture du T-Beam SUPREME

### Laboratoire (3h)
- Remise des projets de mi-session
- Flash du firmware Meshtastic sur T-Beam
- Configuration de base (région, canaux)
- Premier test de communication entre deux noeuds

## Travaux hors classe
- Lecture sur LoRa et Meshtastic
- Préparation pour les tests de portée
- Exploration de l'interface Meshtastic

Bienvenue dans le cours 243-4J5-LI — Objets connectés!

Ce cours vous permettra de concevoir, programmer et déployer des systèmes IoT complets, de la conception de circuits imprimés jusqu'à l'intégration d'intelligence artificielle.

Ce que vous apprendrez :

  • Communication sans fil — MQTT, LoRa, WiFi, LTE
  • Conception électronique — schémas, PCB avec KiCad, soudure
  • Programmation embarquée — ESP32, Arduino, capteurs et actionneurs
  • Réseaux maillés — Meshtastic, communication longue portée
  • Intégration IA — API LLM pour l'analyse intelligente des données

Matériel utilisé :

  • LilyGO A7670G — microcontrôleur ESP32 avec modem LTE
  • T-Beam SUPREME — radio LoRa avec GPS
  • Raspberry Pi — serveur relais et broker MQTT

Évaluations :

  • Laboratoire capteurs et Python — 15%
  • Projet de mi-session (PCB) — 20%
  • TP Intégration LLM — 20%
  • Examen théorique — 15%
  • Projet final — 30%

Conseil : Consultez régulièrement le résumé de chaque semaine pour voir les tâches à faire et les prochaines évaluations.

## Utilisation

Chaque fichier contient un bloc `<div>` HTML avec le résumé de la semaine, formaté pour Moodle. Il suffit de :

1. Ouvrir le fichier HTML de la semaine
2. Copier tout le contenu
3. Coller dans l'éditeur Moodle en mode HTML

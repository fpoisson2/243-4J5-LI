Mise en pratique de Meshtastic sur les T-Beam SUPREME : flash du firmware, connexion aux téléphones et tests de portée terrain avec collecte de métriques radio.

Mise en route :

  • Flash du firmware — via le Web Flasher (flasher.meshtastic.org)
  • Connexion Bluetooth — appairage avec l'application Meshtastic (Android/iOS)
  • Configuration — nom du noeud, région US (915 MHz), canal de classe
  • Preset MEDIUM_FAST — bon compromis portée/débit pour les tests

Tests de portée :

  • RSSI — puissance du signal reçu (dBm), plus proche de 0 = plus fort
  • SNR — rapport signal/bruit (dB), LoRa fonctionne même en SNR négatif
  • Protocole — équipes de 2, un fixe + un mobile, mesures tous les 50-100 m
  • Environnements — intérieur (couloirs), extérieur (terrain dégagé)

Fonctionnalités explorées :

  • Rôles — CLIENT, ROUTER, REPEATER, TRACKER, SENSOR
  • Télémétrie — batterie, utilisation radio, position GPS
  • Traceroute — visualiser le chemin multi-hop d'un message

Labo : flasher le firmware Meshtastic, connecter l'app au téléphone, configurer le réseau de classe et effectuer des tests de portée en extérieur.

À faire pour la semaine prochaine :

  • S'assurer que le firmware Meshtastic est fonctionnel sur votre T-Beam
  • Avoir un compte API LLM prêt (Groq, OpenAI ou Anthropic)
  • Revoir vos résultats de tests de portée (RSSI, SNR, distances)

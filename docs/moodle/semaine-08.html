Introduction a la technologie LoRa (Long Range) pour la communication longue portee sans infrastructure, et decouverte de Meshtastic pour creer un reseau mesh decentralise.

Technologie LoRa :

  • Modulation CSS -- Chirp Spread Spectrum, resistante aux interferences
  • Portee -- 2 a 15 km selon les parametres et l'environnement
  • Bande 915 MHz -- ISM sans licence en Amerique du Nord
  • Compromis -- longue portee mais faible debit (0.3 a 50 kbps)

Meshtastic :

  • Firmware open source -- transforme des radios LoRa en reseau mesh
  • Reseau decentralise -- aucune infrastructure requise
  • Chiffrement AES-256 -- communication securisee
  • T-Beam SUPREME -- ESP32-S3 + LoRa SX1262 + GPS

Parametres radio :

  • Spreading Factor (SF7-SF12) -- plus eleve = plus de portee, moins de debit
  • Bandwidth (125-500 kHz) -- plus large = plus de debit, moins de portee
  • Coding Rate (4/5 a 4/8) -- redondance pour corriger les erreurs
  • Presets Meshtastic -- SHORT_FAST, MEDIUM_FAST, LONG_SLOW, etc.
